# Claude Code Configuration

Claude Code configuration for projects. Contains custom skills, hooks, and settings.

### Third-Party Skills

When installing a skill from an external repo:

1. **Install verbatim** — do not trim, rewrite, or modify third-party skill files. Use them exactly as provided by the author.
2. **Document in README.md** — add the skill name, source repo URL, and a short description to the **Third-Party Skills** table.
